# Hyde Park MRO — Part Interchange Recommender

Given a manufacturer part number, returns ranked alternative parts from other brands in the catalog, with confidence tiers (Green / Yellow / Red).

## Setup

```bash
conda env create -f environment.yml
conda activate mro-recommender
```

## Run

```bash
# 1. Clean raw catalog
python scripts/clean_data.py          # data/raw_data.csv → data/cleaned_data.csv

# 2. Build the model index (~2–3 min)
python scripts/build_index.py         # → models/recommender.joblib

# 3. Query
python scripts/query.py --brand "Humphrey Products" --part "E3P"
python scripts/query.py --brand Parker --part "P32EA4510AABW" --top-n 5
python scripts/query.py --search-brand "Parker"
python scripts/query.py --search-parts --brand "Humphrey Products" --part "E3"
```

## How It Works

1. **Text normalization** — synonyms collapsed to canonical tokens (`"palm button"` → `"push-button"`, `"parker hannifin"` → `"Parker"`)
2. **TF-IDF index** — built over all 292k parts, filtered at query time to same category / attribute family
3. **Confidence scoring** — hybrid: shared canonical spec attributes (primary) + cosine similarity (secondary)
   - **Green** — 3+ shared spec attributes, same category
   - **Yellow** — 2 shared spec attributes
   - **Red** — same category, partial or no spec overlap
